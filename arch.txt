File Description
- main.ino: contains the main loop that calls all sensors, runs algorithm, calls actuators
- sensors: basically all analog in description of the location of the sensor
- actuator: basically an anlog out (using pwm lib) and loaction 
- mux: utility that makes accessing the mux easier
- algorithms: classes that contain bdot/modified bdot algorithm
